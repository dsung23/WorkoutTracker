# Workout Tracker

A Simple full-stack application to track your workouts.

## Features

- Add, view, and delete workouts
- Responsive UI
- RESTful API

## Tech Stack

- Backend: Node.js, Express, MongoDB
- Frontend: React

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
